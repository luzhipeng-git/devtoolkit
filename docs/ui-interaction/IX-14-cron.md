# IX-14 Cron 表达式交互流程

> 页面文件：`14-cron.html`
> 功能模块：Cron 表达式编辑与解析

---

## 界面状态定义

| 状态名称 | 触发条件 | 界面表现 |
|---------|---------|---------|
| 初始态 | 页面首次加载 | 页面分为左右两栏。**左栏**为可视化编辑区：顶部为字段标签页（秒/分/时/日/月/周/年），默认选中"分"标签；标签页下方为当前字段的模式单选按钮组和对应的编辑控件；**右栏**为预览区：顶部为 Cron 表达式输入框（等宽字体，默认 `*/5 * * * *`）+ 收藏/常用按钮；中间为各字段的分解显示（秒、分、时、日、月、周、年各自对应的值）；表达式下方为人类可读中文描述；底部为"最近 10 次运行时间"列表。系统以默认表达式初始化所有界面元素 |
| 编辑态 | 用户正在修改 Cron 表达式或可视化编辑器 | 表达式输入框内容实时更新；右栏各字段分解区实时同步；人类可读描述实时同步；"最近 N 次执行时间"实时重新计算并更新列表 |
| 结果态 | Cron 表达式合法且稳定 | 人类可读描述清晰展示执行规则；"最近 N 次执行时间"显示未来 5-10 次精确执行时间；表达式输入框无错误边框 |
| 异常态 | Cron 表达式语法错误或语义无效 | 表达式输入框边框变红并在下方显示具体错误信息（如"第 3 个字段无效：日期范围为 1-31"）；人类可读描述替换为错误提示；"最近 N 次执行时间"区域清空并显示"表达式无效，无法计算执行时间"；可视化编辑器中出错字段高亮标红 |

### 可视化编辑器模式选项（按字段类型）

每个字段标签页下方提供一组模式单选按钮，不同字段的模式选项有所不同：

**秒（0-59）、分（0-59）**

| 模式 | 单选按钮文案 | 编辑控件 | 生成的 Cron 值 |
|------|-------------|---------|---------------|
| 每 | "每秒" / "每分" | 无 | `*` |
| 步长 | "周期：从 [X] 开始，每 [Y] 秒/分执行一次" | 两个数字输入框（带上下箭头） | `X/Y` |
| 范围 | "周期：从 [X] - [Y] 秒/分" | 两个数字输入框（带上下箭头） | `X-Y` |
| 指定 | "指定：" | 复选框网格（0-59，10 列 × 6 行） | `a,b,c,...` |

**时（0-23）**

| 模式 | 单选按钮文案 | 编辑控件 | 生成的 Cron 值 |
|------|-------------|---------|---------------|
| 每 | "每小时" | 无 | `*` |
| 步长 | "周期：从 [X] 开始，每 [Y] 小时执行一次" | 两个数字输入框 | `X/Y` |
| 范围 | "周期：从 [X] - [Y] 小时" | 两个数字输入框 | `X-Y` |
| 指定 | "指定：" | 复选框网格（0-23，6 列 × 4 行） | `a,b,c,...` |

**日（1-31）**

| 模式 | 单选按钮文案 | 编辑控件 | 生成的 Cron 值 |
|------|-------------|---------|---------------|
| 每 | "每天" | 无 | `*` |
| 不指定 | "不指定" | 无 | `?` |
| 步长 | "周期：从 [X] 开始，每 [Y] 天执行一次" | 两个数字输入框 | `X/Y` |
| 范围 | "周期：从 [X] - [Y] 日" | 两个数字输入框 | `X-Y` |
| 最近工作日 | "每月 [X] 日之后最近的那个工作日" | 一个数字输入框 | `XW` |
| 最后一天 | "每月最后一天" | 无 | `L` |
| 指定 | "指定：" | 复选框网格（1-31，10 列 × 4 行） | `a,b,c,...` |

**月（1-12）**

| 模式 | 单选按钮文案 | 编辑控件 | 生成的 Cron 值 |
|------|-------------|---------|---------------|
| 每 | "每月" | 无 | `*` |
| 步长 | "周期：从 [X] 开始，每 [Y] 月执行一次" | 两个数字输入框 | `X/Y` |
| 范围 | "周期：从 [X] - [Y] 月" | 两个数字输入框 | `X-Y` |
| 指定 | "指定：" | 复选框网格（1-12，6 列 × 2 行） | `a,b,c,...` |

**周（1-7 或 SUN-SAT）**

| 模式 | 单选按钮文案 | 编辑控件 | 生成的 Cron 值 |
|------|-------------|---------|---------------|
| 不指定 | "不指定" | 无 | `?` |
| 步长 | "周期：从第 [X] 周开始，每 [Y] 周执行一次" | 两个数字输入框 | `X/Y` |
| 范围 | "周期：第 [X] - [Y] 周" | 两个数字输入框 | `X-Y` |
| 指定 | "指定：" | 复选框网格（1-7，7 列 × 1 行，标签显示 SUN-SAT） | `a,b,c,...` |
| 第 N 周星期 M | "第 [N] 周的星期 [M]" | 两个数字输入框 | `NW` 或 `N#M` |

**年（可选字段）**

| 模式 | 单选按钮文案 | 编辑控件 | 生成的 Cron 值 |
|------|-------------|---------|---------------|
| 不指定 | "不指定" | 无 | （空，不生成年字段） |
| 步长 | "周期：从 [X] 年开始，每 [Y] 年执行一次" | 两个数字输入框 | `X/Y` |
| 范围 | "周期：从 [X] - [Y] 年" | 两个数字输入框 | `X-Y` |
| 指定 | "指定：" | 复选框网格（当前年份前后若干年） | `a,b,c,...` |

## 操作流程

1. 页面加载后，系统以默认 Cron 表达式"*/5 * * * *"初始化所有界面元素。
2. 系统将默认表达式解析为人类可读的中文描述，显示在右栏表达式下方。
3. 用户可直接在右栏的表达式输入框中编辑 Cron 表达式，每次输入变化触发实时解析和同步更新。
4. 用户可切换左栏的可视化编辑标签页（秒/分/时/日/月/周/年），切换后显示对应字段的模式单选按钮和编辑控件。
5. 每个字段的模式选项见上方"可视化编辑器模式选项"表格。用户选择某个模式后，下方显示对应的编辑控件：
   - **"每"模式**：无需额外输入，直接生成通配符值。
   - **"步长"模式**：显示两个数字输入框（起始值、步长值），带上下箭头按钮调整。
   - **"范围"模式**：显示两个数字输入框（起始值、结束值），带上下箭头按钮调整。
   - **"指定"模式**：显示复选框网格，用户勾选多个具体数值，已勾选项以蓝色高亮。
   - **"不指定"模式**：无需额外输入，生成 `?` 值（仅日和周字段可用）。
   - **"最近工作日"模式**：显示一个数字输入框指定日期（仅日字段可用）。
   - **"最后一天"模式**：无需额外输入，生成 `L` 值（仅日字段可用）。
6. 用户在可视化编辑器中修改任何字段，系统立即更新右栏的表达式输入框内容，同步更新人类可读描述、各字段分解显示和执行时间列表（可视化→表达式同步）。
7. 用户在表达式输入框中手动修改内容，系统立即更新可视化编辑器对应字段的模式选择状态和控件值，并同步更新人类可读描述和执行时间列表（表达式→可视化同步）。
8. 用户点击右栏的"常用 Cron"按钮，展开预设模板列表（如"每分钟"、"每小时"、"每天零点"、"每周一"、"每月 1 日零点"、"工作日早 9 点"等），点击某项后系统将对应表达式填入输入框并触发全面更新。
9. 用户点击右栏的"收藏"按钮，将当前表达式保存到收藏夹；点击"收藏夹"按钮可查看和管理已收藏的表达式。
10. 系统根据当前表达式实时计算并展示未来 10 次执行时间，每次表达式变更均重新计算。
11. 用户点击任意执行时间旁的复制按钮，将对应时间字符串复制到系统剪贴板。

## 交互规则

| 交互动作 | 触发元素 | 响应行为 | 去重保护 | 约束条件 |
|---------|---------|---------|---------|---------|
| 编辑表达式 | 右栏表达式输入框（`input` 事件） | 实时解析表达式语法；合法时更新人类可读描述、可视化编辑器、各字段分解显示、执行时间列表；非法时进入异常态 | — | 输入框使用等宽字体，字母间距加大以增强可读性；支持 5 字段（分时日月周）和 7 字段（秒分时日月周年）两种格式 |
| 切换编辑标签页 | 左栏 秒/分/时/日/月/周/年 标签 | 切换可视化编辑器显示对应字段的模式单选按钮和编辑控件，保留其他字段的当前值 | **重复点击当前已激活的标签页则忽略** | 标签页切换不丢失任何字段的编辑状态 |
| 切换模式单选按钮 | 字段模式单选按钮（每/步长/范围/指定/不指定 等） | 切换到对应模式的编辑控件界面；若新模式生成了与旧模式不同的 Cron 值，则同步更新表达式输入框 | **重复选择当前已激活的模式则忽略** | 模式切换保留该字段之前的模式选择状态（切回时可恢复） |
| 步长模式数字输入 | 步长模式的起始值/步长值数字输入框（带上下箭头） | 修改数字后立即生成新的 `X/Y` 值，更新表达式输入框并同步触发全面更新 | —（数字输入框） | 起始值和步长值均需在字段有效范围内 |
| 范围模式数字输入 | 范围模式的起始值/结束值数字输入框（带上下箭头） | 修改数字后立即生成新的 `X-Y` 值，更新表达式输入框并同步触发全面更新 | —（数字输入框） | 起始值不得大于结束值 |
| 指定模式勾选复选框 | 指定模式的复选框网格 | 勾选/取消勾选某个数值后，立即将所有已勾选值拼接为逗号分隔的字符串（如 `5,14,15`），更新表达式输入框并同步触发全面更新 | —（复选框操作） | 至少勾选一个值；若全部取消勾选则自动回退到"每"模式 |
| 日字段特殊模式 | "不指定" / "最近工作日" / "最后一天" 单选按钮 | 不指定：生成 `?`；最近工作日：读取数字输入框生成 `XW`；最后一天：生成 `L`；选择后同步更新表达式输入框 | **重复选择当前已激活的模式则忽略** | 日字段选"不指定"时，周字段不能同时为 `?`（两者必须有一个不是 `?`） |
| 表达式→可视化同步 | 表达式输入框内容变化 | 解析表达式各字段值，将每个字段映射为可视化编辑器中的对应模式（如 `5,14,15` 映射为"指定"模式并勾选对应复选框，`0/1` 映射为"步长"模式并填入数字） | — | 解析失败时可视化编辑器保持上次有效状态并标红出错字段 |
| 可视化→表达式同步 | 可视化编辑器内容变化 | 根据各字段的可视化选择结果拼接为完整的 Cron 表达式字符串并填入输入框，同步更新各字段分解显示 | — | — |
| 点击预设模板 | "常用 Cron"按钮展开的模板列表项 | 将模板对应的表达式填入输入框，触发表达式→可视化→描述→执行时间的全面更新 | —（模板为便捷按钮，每次点击有明确意图） | 常用模板：每分钟（`* * * * *`）、每小时（`0 * * * *`）、每天零点（`0 0 * * *`）、每周一零点（`0 0 * * 1`）、每月 1 日零点（`0 0 1 * *`）、工作日早 9 点（`0 9 * * 1-5`） |
| 收藏表达式 | "收藏"按钮 | 将当前 Cron 表达式和中文描述保存到收藏夹列表 | — | 收藏夹持久化到 localStorage，最多保留 20 条 |
| 管理收藏夹 | "收藏夹"按钮 | 打开收藏夹弹窗，显示已收藏的表达式列表；每项显示表达式、描述、收藏时间；支持点击填入和删除 | — | 点击收藏项将表达式填入输入框并触发全面更新 |
| 复制执行时间 | 执行时间旁的复制按钮 | 将对应时间字符串写入系统剪贴板，按钮显示"已复制"反馈 | — | 反馈持续 2 秒后恢复 |
| 复制表达式 | 表达式旁的复制按钮 | 将当前 Cron 表达式写入系统剪贴板 | — | — |
| 错误定位 | 表达式语法错误 | 在错误信息中标注具体出错字段的序号和位置，对应标签页标红 | — | 支持的错误类型：字段数不正确、数值超出范围、非法字符、不支持的操作符、日和周同时为 `?` |
| 各字段分解显示 | 右栏中间区域 | 以标签+值的行形式展示 Cron 表达式各字段（秒：24,33 / 分：0/1 / 时：* / 日：5,14,15 / 月：* / 周：? / 年：），每个字段值独立显示 | — | 与表达式输入框双向同步，表达式变化时立即更新 |
