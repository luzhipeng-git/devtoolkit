# IX-01 主框架交互流程

> 对应页面：`01-main-framework.html`
> 覆盖范围：应用主窗口框架、侧栏导航、搜索覆盖层、内容区域

---

## 一、界面状态定义

| 状态名称 | 触发条件 | 界面表现 |
|---------|---------|---------|
| 初始态 | 应用启动，未选择任何工具 | 侧栏展开，内容区显示占位页面（快捷键提示 + 功能卡片网格），面包屑显示"首页" |
| 工具态 | 用户点击侧栏工具项或搜索结果 | 侧栏对应项高亮，面包屑更新为"分类 > 工具名"，内容区加载工具页面 |
| 搜索态 | 用户按下 `Ctrl+K` | 搜索覆盖层弹出，背景半透明遮罩，搜索输入框自动聚焦，显示分组搜索结果 |
| 侧栏折叠态 | 用户按下 `Ctrl+B` 或点击折叠按钮 | 侧栏宽度收窄为图标列（仅显示分类图标），内容区扩展占满剩余宽度 |

## 二、操作流程

### 2.1 应用启动流程

1. 渲染主窗口框架（标题栏 + 侧栏 + 内容区）
2. 侧栏默认展开，加载完整分类与工具列表
3. 内容区进入 **初始态**，显示占位页面
4. 占位页面包含：快捷键提示区域（`Ctrl+K` 搜索、`Ctrl+B` 切换侧栏）+ 功能卡片网格（展示常用工具入口）
5. 面包屑显示"首页"

### 2.2 侧栏导航流程

1. 用户在侧栏点击分类名称 → 该分类展开，显示子工具列表；再次点击则折叠
2. 用户点击子工具项 → 界面进入 **工具态**：
   - 侧栏该工具项高亮（背景色变化）
   - 面包屑更新为"分类名 > 工具名"
   - 内容区加载对应工具页面
3. 用户点击另一个工具 → 替换当前内容区，更新高亮和面包屑

### 2.3 搜索覆盖层流程

1. 用户按下 `Ctrl+K`（或点击侧栏搜索框）→ 界面进入 **搜索态**：
   - 显示半透明遮罩覆盖全屏
   - 搜索覆盖层弹出，输入框自动获得焦点
   - 初始显示全部工具（按分类分组）
2. 用户输入关键词 → 实时过滤结果：
   - 同时匹配工具名称和所属分类名
   - 结果按分类分组展示
   - 无匹配时显示"未找到相关工具"提示
3. 用户使用 `↑` `↓` 键盘方向键 → 在结果列表中移动焦点高亮
4. 用户按下 `Enter` → 加载当前高亮的工具，关闭搜索覆盖层，进入 **工具态**
5. 用户按下 `Esc`（或点击遮罩背景） → 关闭搜索覆盖层，回到之前的状态

### 2.4 侧栏折叠/展开流程

1. 用户按下 `Ctrl+B`（或点击折叠按钮）→ 界面进入 **侧栏折叠态**：
   - 侧栏宽度收窄为仅显示分类图标
   - 分类名称和子工具列表隐藏
   - 内容区水平扩展占满空间
2. 用户再次按下 `Ctrl+B`（或点击展开按钮）→ 恢复侧栏展开：
   - 侧栏滑出完整列表
   - 内容区宽度恢复
3. 鼠标悬停在折叠态侧栏的分类图标上 → 显示 tooltip 分类名称

### 2.5 占位页面（初始态内容区）交互

1. 显示快捷键提示：
   - `Ctrl+K`：快速搜索工具
   - `Ctrl+B`：切换侧栏
2. 显示功能卡片网格：
   - 每张卡片包含工具图标 + 工具名称 + 简要描述
   - 响应式布局：默认 4 列 → 中等屏幕 3 列 → 小屏幕 2 列
3. 用户点击功能卡片 → 进入 **工具态**，加载对应工具页面

## 三、交互规则

### 3.1 快捷键规则

| 快捷键 | 作用域 | 行为 |
|--------|--------|------|
| `Ctrl+K` | 全局 | 打开搜索覆盖层；若已打开则关闭 |
| `Ctrl+B` | 全局 | 切换侧栏折叠/展开状态 |
| `Esc` | 搜索覆盖层内 | 关闭搜索覆盖层 |
| `↑` / `↓` | 搜索覆盖层内 | 在搜索结果中上下移动焦点 |
| `Enter` | 搜索覆盖层内 | 确认选择当前高亮的工具 |

### 3.2 搜索过滤规则

1. 搜索关键词同时匹配工具名称和所属分类名称，不区分大小写
2. 结果按分类分组，每个分类下按原始顺序排列
3. 输入为空时显示全部工具列表
4. 无匹配结果时显示空状态提示文案"未找到相关工具"

### 3.3 导航与高亮规则

1. 侧栏当前选中工具项始终高亮显示
2. 切换工具时，旧工具高亮立即取消，新工具高亮立即生效
3. 面包屑路径格式固定为"分类名 > 工具名"
4. 点击面包屑中的分类名 → 滚动到首页该分类区域（若实现首页功能）

### 3.4 侧栏折叠规则

1. 折叠动画平滑过渡，时长不超过 200ms
2. 折叠后仅保留分类图标列，宽度约 48-56px
3. 展开后恢复完整侧栏，宽度约 240-280px
4. 折叠态下的当前选中工具，其所属分类图标需有视觉指示（如背景色或边框）

### 3.5 内容区加载规则

1. 工具页面加载时，内容区替换为新工具内容，无页面刷新
2. 加载过程中可显示轻量加载指示器（如旋转图标）
3. 每个工具页面独立管理自身状态（输入值、选项等），切换离开后再返回应恢复上次状态
4. 占位页面仅在未选择任何工具时显示

### 3.6 搜索覆盖层交互规则

1. 覆盖层打开时，背景遮罩透明度约 50%，阻止对下方内容的点击
2. 点击遮罩区域（覆盖层外部）等同于按 `Esc`，关闭覆盖层
3. 覆盖层打开/关闭使用淡入淡出动画，时长不超过 150ms
4. 覆盖层关闭后，搜索框内容清空，下次打开时为空输入状态
