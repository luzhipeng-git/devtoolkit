## 一、界面状态定义

## 二、操作流程

### 2.1 证书解析流程

1. 用户在「证书解析」Tab 下，将 PEM 格式证书粘贴到输入 textarea，或通过文件上传按钮选择证书文件（.pem / .cer / .crt / .der）
2. 用户点击「解析」按钮
3. 系统校验输入：
   - 检查 PEM 格式标记（-----BEGIN CERTIFICATE----- / -----END CERTIFICATE-----）
   - 检查 Base64 编码合法性
4. 校验通过后，系统解析证书字段
5. 解析结果以树形结构展示：
   - **使用者（Subject）**：CN、O、OU、C、ST、L 等字段
   - **颁发者（Issuer）**：CA 的 Subject 信息
   - **有效期（Validity）**：生效时间、过期时间、剩余天数（以颜色标识：绿色 > 30 天、黄色 7-30 天、红色 < 7 天）
   - **序列号（Serial Number）**：Hex 格式
   - **签名算法**：如 SHA256withRSA
   - **公钥信息**：算法、密钥长度、公钥值
   - **指纹**：SHA-1、SHA-256 指纹
   - **扩展项（Extensions）**：SAN、密钥用途、基本约束等
6. 每个字段旁有复制按钮，可单独复制字段值

### 2.2 CSR 生成流程

1. 用户切换到「CSR 生成」Tab
2. 用户填写 CSR 参数表单：
   - 通用名（CN）：必填，通常为域名
   - 组织（O）：可选
   - 组织单元（OU）：可选
   - 国家（C）：必填，两位国家代码，下拉选择
   - 省/州（ST）：可选
   - 地区（L）：可选
   - 密钥长度：2048（默认）/ 4096
3. 用户点击「生成 CSR」按钮
4. 系统校验必填字段（CN、C）
5. 使用 Web Crypto API 生成密钥对并构建 CSR
6. 输出区显示 PEM 格式 CSR（readonly textarea），可复制
7. 私钥输出区显示 PEM 格式私钥（默认遮罩），可切换可见性和复制

### 2.3 格式转换流程

1. 用户切换到「格式转换」Tab
2. 用户在输入区粘贴证书/密钥内容，或通过文件上传按钮选择文件
3. 用户选择源格式（PEM / DER / P7B / PFX）和目标格式
4. 用户点击「转换」按钮
5. 系统校验输入内容与源格式是否匹配
6. 执行格式转换
7. 若目标格式为 PEM，输出区显示 PEM 文本 + 复制按钮
8. 若目标格式为 DER/P7B/PFX（二进制），提供下载按钮

### 2.4 命令构建流程

1. 用户切换到「命令构建」Tab
2. 用户从命令类型下拉框中选择操作类型：
   - 生成自签名证书
   - 生成 CSR
   - 查看证书信息
   - 转换证书格式
   - 生成私钥
   - 验证证书
3. 根据所选操作类型，动态显示对应参数表单：
   - 每个参数有标签、输入框/下拉框、默认值
   - 参数变更实时更新下方命令预览
4. 命令预览区以终端样式（深色背景、等宽字体）显示完整 OpenSSL 命令
5. 命令字符串支持一键复制
6. 用户可随时修改参数，预览区即时更新

## 三、交互规则

| 规则编号 | 触发条件 | 行为描述 | 去重保护 | 约束说明 |
|---------|---------|---------|---------|---------|
| R-09-01 | 主 Tab 切换（证书解析/CSR 生成/格式转换/命令构建） | 切换到对应 Tab 内容区；保留各 Tab 内已有的输入数据和结果 | **重复点击当前已激活的 Tab 则忽略** | Tab 切换不丢失任何已输入数据 |
| R-09-02 | 证书 PEM 输入校验 | 用户输入时实时检查 PEM 标记；缺少 BEGIN/END 标记时 textarea 边框变红并显示提示「请输入包含完整 PEM 标记的证书」 | —（输入校验，不涉及选择型控件） | 实时反馈，不阻断输入 |
| R-09-03 | 点击「解析」按钮 | 校验 PEM 格式和 Base64 编码；校验通过执行解析；校验失败显示具体错误信息 | —（操作按钮，不涉及选择型控件） | 解析在前端完成，不发送到服务器 |
| R-09-04 | 证书解析结果树形展示 | 以可折叠树形结构展示；一级节点默认展开；二级节点（如扩展项的子项）默认折叠；点击节点标题可展开/折叠 | —（展示规则，不涉及用户主动选择） | 使用缩进和连接线表示层级关系 |
| R-09-05 | 有效期剩余天数颜色 | > 30 天：绿色；7-30 天：黄色；< 7 天：红色；已过期：红色 + 「已过期」标签 | —（展示规则，不涉及用户主动选择） | 颜色应用到剩余天数文字上 |
| R-09-06 | 解析结果字段复制 | 每个字段旁有复制图标按钮；点击后将字段值复制到剪贴板；按钮短暂变为勾选 + 「已复制」 | —（操作按钮，不涉及选择型控件） | 单个字段独立复制 |
| R-09-07 | CSR 表单必填校验 | CN 和 C 为必填；点击「生成 CSR」时校验；未填写时对应输入框边框变红并显示「此字段为必填项」 | —（校验规则，不涉及选择型控件） | 校验在点击时触发 |
| R-09-08 | CSR 国家代码输入 | 使用下拉选择框，包含常用国家代码（CN、US、JP、GB、DE、FR 等）；选择后自动填入两位代码 | —（下拉选择，但作为表单子字段，不涉及主控切换） | 仅允许选择，不支持手动输入 |
| R-09-09 | CSR 生成结果 | CSR 输出区为 readonly textarea，显示完整 PEM 格式 CSR；私钥输出区默认遮罩，可切换可见性；各自有独立复制按钮 | —（展示规则，不涉及用户主动选择） | CSR 和私钥分别展示，互不影响 |
| R-09-10 | 格式转换源格式选择 | 选择源格式后，输入区提示更新（如「请粘贴 PEM 格式证书」或「请上传 DER 格式文件」）；DER/PFX/P7B 格式同时显示文件上传按钮 | **重复选择当前源格式则忽略** | PEM 支持文本粘贴，二进制格式仅支持文件上传 |
| R-09-11 | 格式转换目标格式智能推荐 | 选择源格式后，目标格式下拉框自动排除与源格式相同的选项；常用转换路径排在前面 | —（智能推荐，不涉及用户主动选择） | 避免无意义的同格式转换 |
| R-09-12 | 格式转换输出 | 目标为 PEM：textarea 显示 + 复制按钮；目标为 DER/P7B/PFX：显示下载按钮，文件名自动生成（cert_converted.der） | —（展示规则，不涉及用户主动选择） | 二进制格式不尝试文本展示 |
| R-09-13 | 命令构建类型切换 | 切换操作类型时，参数表单动态替换为新类型的参数；命令预览区同步更新；之前类型填写的参数不保留 | **重复选择当前命令类型则忽略**，不重复销毁和重建参数表单 | 每种类型有独立的参数集合 |
| R-09-14 | 命令预览实时更新 | 用户修改任一参数时，命令预览区立即重新生成完整命令字符串；参数值为空时使用 OpenSSL 默认值或省略该选项 | —（参数输入，不涉及选择型控件） | 使用 input/change 事件实时触发 |
| R-09-15 | 命令预览样式 | 深色背景（#1e1e1e）、浅色等宽字体；命令关键字（openssl）加粗高亮；参数名（-x509、-nodes 等）使用不同颜色 | —（展示规则，不涉及用户主动选择） | 仿终端样式，提高可读性 |
| R-09-16 | 命令复制 | 点击复制按钮将完整命令字符串复制到剪贴板；按钮变为勾选 + 「已复制」；2 秒后恢复 | —（操作按钮，不涉及选择型控件） | 复制纯文本，不含样式 |
| R-09-17 | 文件上传大小限制 | 上传文件大小限制为 1MB；超过时显示错误「文件过大，最大支持 1MB」 | —（校验规则，不涉及选择型控件） | 证书文件通常 < 10KB，1MB 足够 |
| R-09-18 | 证书解析文件上传 | 证书解析输入区支持文件上传，点击「上传文件」按钮后打开系统文件选择对话框；支持的文件格式：.pem、.cer、.crt、.der；PEM 格式文件自动读取文本内容填入 textarea；DER 格式（二进制）文件自动转为 Base64 后填入 | —（操作按钮，不涉及选择型控件） | 文件读取后自动触发 PEM 格式校验 |
