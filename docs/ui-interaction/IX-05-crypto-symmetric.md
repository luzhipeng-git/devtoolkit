# IX-05 对称加密交互流程

> 涵盖页面：`05-crypto-aes.html`（AES）、`05b-crypto-des.html`（DES/3DES）

---

## 一、界面状态定义

| 状态名称 | 说明 | 界面表现 |
|---------|------|---------|
| 初始态 | 页面加载完成，用户尚未操作 | 加密 Tab 默认选中；模式 CBC；填充 PKCS7；密钥长度 128（AES）/ 单倍长 DES（DES）；输出格式 Base64；IV 可见开关打开；密钥输入为空；IV 输入为空；输入编辑器为空；输出编辑器显示占位文本 |
| 加密态 | 当前 Tab 为加密 | 输入标签显示「明文」；输出标签显示「密文」；「加密」操作按钮可用 |
| 解密态 | 当前 Tab 为解密 | 输入标签显示「密文」；输出标签显示「明文」；「解密」操作按钮可用 |
| ECB 模式态 | 加密模式切换为 ECB | IV 输入行隐藏（display:none）；IV 生成按钮隐藏；IV 不参与加解密计算 |
| 非 ECB 模式态 | 加密模式为 CBC/GCM/CTR | IV 输入行可见；IV 生成按钮可见；IV 参与加解密计算 |
| 密钥隐藏态 | 密钥输入默认状态 | 密钥输入框内容以圆点遮罩显示（CSS text-security:disc）；眼睛图标为「闭眼」状态 |
| 密钥可见态 | 用户点击眼睛图标后 | 密钥输入框内容以明文 Hex 显示；眼睛图标切换为「睁眼」状态 |
| IV 隐藏态 | IV 可见开关关闭 | IV 输入行整体隐藏；加解密使用内部 IV（若已生成）或随机生成 |
| IV 可见态 | IV 可见开关打开 | IV 输入行可见；用户可手动输入或自动生成 IV |
| 执行中态 | 用户点击加密/解密按钮后 | 操作按钮显示 loading spinner 并禁用；输入区域不可编辑；输出编辑器显示「处理中...」 |
| 结果态 | 加密/解密完成 | 输出编辑器显示结果（Base64 或 Hex 格式）；复制按钮可用；历史记录更新 |
| 异常态 | 输入不合法或加解密失败 | 输入框下方显示红色错误提示；操作按钮恢复可用；输出编辑器清空 |
| 已交换态 | 用户点击交换按钮后 | 原输出内容填入输入编辑器；原输入内容填入输出编辑器；Tab 自动切换（加密↔解密） |

## 二、操作流程

### 2.1 AES 加密流程

1. 用户在「加密」Tab 下，确认或修改加密参数（模式、填充、密钥长度、输出格式）
2. 用户在密钥输入框输入 Hex 格式密钥，或点击「随机生成」按钮生成密钥
   - 128 位密钥生成 16 字节（32 个 Hex 字符）
   - 192 位密钥生成 24 字节（48 个 Hex 字符）
   - 256 位密钥生成 32 字节（64 个 Hex 字符）
3. 若模式非 ECB，用户输入或点击生成 IV（始终 16 字节 / 32 个 Hex 字符）
4. 用户在输入编辑器中输入明文
5. 用户点击「加密」按钮
6. 界面进入执行中态，调用 Web Crypto API 执行 AES 加密
7. 加密完成后，输出编辑器显示密文（按所选输出格式编码）
8. 历史记录列表新增一条记录，包含：时间戳、模式、密钥长度、输入摘要

### 2.2 AES 解密流程

1. 用户切换到「解密」Tab → 若当前已在「解密」Tab 则忽略操作（去重）
2. 确认或修改解密参数（需与加密时一致）
3. 输入密钥和 IV（需与加密时一致）
4. 在输入编辑器中粘贴密文（Base64 或 Hex 格式，需与输出格式设置一致）
5. 点击「解密」按钮
6. 界面进入执行中态，执行 AES 解密
7. 解密成功后，输出编辑器显示明文
8. 解密失败（密钥错误、密文损坏等）则进入异常态，显示错误信息

### 2.3 输入交换流程

1. 用户点击输入/输出编辑器之间的「交换」按钮（双向箭头图标）
2. 系统将当前输出内容复制到输入编辑器
3. 系统将当前输入内容复制到输出编辑器
4. 当前 Tab 自动切换（加密↔解密）
5. 交换后用户可直接执行反向操作

### 2.4 DES/3DES 操作流程

1. 用户选择密钥类型：单倍长 DES（8 字节）/ 双倍长 3DES（16 字节）/ 三倍长 3DES（24 字节）
2. 密钥长度根据密钥类型自动确定，不可手动修改
3. IV 固定 8 字节（DES 块大小），仅在非 ECB 模式下可见
4. 其余操作流程与 AES 一致

## 三、交互规则

| 规则编号 | 触发条件 | 行为描述 | 去重保护 | 约束说明 |
|---------|---------|---------|---------|---------|
| R-05-01 | Tab 切换（加密↔解密） | 更新输入/输出标签文字（明文↔密文）；清空输出编辑器内容；保留输入编辑器和密钥/IV 内容不变 | **点击当前已激活的 Tab 则忽略**，不做任何状态变更或内容清空 | 切换不丢失用户已输入的数据 |
| R-05-02 | 模式切换为 ECB | 隐藏 IV 输入行（含标签、输入框、生成按钮）；清空已有 IV 值 | **重复选择当前已是 ECB 的模式则忽略**，不重复清空 IV | 仅 ECB 模式隐藏 IV，其余模式均显示 |
| R-05-03 | 模式从 ECB 切换为其他 | 显示 IV 输入行；IV 输入为空，需用户手动输入或点击生成 | **重复选择当前已是该模式的选项则忽略** | 切换后不自动生成 IV |
| R-05-04 | 点击「随机生成密钥」 | 根据「密钥长度」选项生成对应字节数的随机 Hex 字符串（使用 crypto.getRandomValues）；填入密钥输入框；输入框闪烁绿色边框表示已更新 | 无需去重（每次点击都生成新的随机值，操作本身幂等） | 生成前无需确认，直接覆盖原有密钥 |
| R-05-05 | 点击「随机生成 IV」 | 生成 16 字节（AES）或 8 字节（DES）随机 Hex 字符串；填入 IV 输入框；输入框闪烁绿色边框 | 无需去重（每次生成新的随机值） | IV 生成按钮仅在 IV 行可见时可用 |
| R-05-06 | 点击密钥眼睛图标 | 切换密钥输入框的显示/隐藏状态；图标在睁眼/闭眼之间切换 | 无需去重（toggle 开关，每次都翻转状态） | 使用 CSS text-security 属性实现遮罩，不改变实际值 |
| R-05-07 | IV 可见开关切换 | 开：显示 IV 输入行；关：隐藏 IV 输入行，加解密时自动使用随机 IV | 无需去重（toggle 开关，每次都翻转状态） | 开关状态不影响已有 IV 值的存储 |
| R-05-08 | 密钥长度变更 | 若当前密钥长度与新选择不匹配，密钥输入框内容清空并显示提示「密钥长度已变更，请重新输入或生成密钥」 | **重复选择当前密钥长度则忽略**，不清空密钥输入框、不显示提示 | 防止使用错误长度密钥 |
| R-05-09 | 点击加密/解密按钮 | 校验：密钥非空、密钥长度合法、（非 ECB 模式时）IV 非空、输入非空；校验通过进入执行中态；校验失败进入异常态，在对应输入框下方显示具体错误信息 | 无需去重（按钮有明确意图，且执行中会禁用） | 所有校验在一次点击中全部执行，显示所有不合规项 |
| R-05-10 | 加密/解密执行成功 | 输出编辑器显示结果；复制按钮激活；结果添加到历史记录（最多保留最近 20 条） | — | 输出格式由「输出格式」选项决定（Base64 或 Hex） |
| R-05-11 | 加密/解密执行失败 | 输出编辑器清空；在输出区域上方显示红色错误横幅，包含错误原因和关闭按钮；操作按钮恢复可用 | — | 错误横幅 5 秒后自动消失，或用户手动关闭 |
| R-05-12 | 点击「交换」按钮 | 交换输入/输出编辑器的内容；自动切换 Tab（加密→解密 或 解密→加密）；清空旧的输出内容 | — | 仅在输出非空时交换按钮可用 |
| R-05-13 | 点击「复制」按钮 | 将输出内容复制到系统剪贴板；按钮图标临时变为勾选状态并显示「已复制」文字；2 秒后恢复原始图标 | — | 使用 navigator.clipboard.writeText API |
| R-05-14 | 历史记录点击 | 将该条记录的输入内容填入输入编辑器；将参数（模式、填充、密钥长度）恢复为记录时的值 | — | 不自动恢复密钥和 IV（安全考虑） |
| R-05-15 | DES 密钥类型切换 | 自动更新密钥长度显示；若当前密钥与新类型长度不匹配，清空密钥输入框并提示重新输入 | **重复选择当前密钥类型则忽略**，不重复清空密钥输入框 | DES: 8 字节, 3DES 双倍: 16 字节, 3DES 三倍: 24 字节 |
| R-05-16 | 输出格式切换 | 若输出编辑器已有结果，自动将结果从旧格式重新编码为新格式显示 | **重复选择当前输出格式则忽略**，不重新编码 | 仅对已有结果进行格式转换，不重新执行加解密 |
| R-05-17 | 填充模式切换 | 无即时效果，下次加密/解密时使用新的填充模式 | **重复选择当前填充模式则忽略** | 切换不触发任何自动操作 |
