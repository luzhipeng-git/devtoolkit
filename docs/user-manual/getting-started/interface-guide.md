# 界面指南

## 整体布局

DevToolkit 采用简洁的双栏布局设计，左侧为工具分类导航，右侧为工具操作区域。

![整体布局](../screenshots/global/layout-overview.png)

- **左侧边栏**: 包含 13 个工具分类，每个分类下有多个具体工具
- **主内容区**: 显示当前选中工具的界面，包含选项配置、输入输出区域和历史记录

## 侧边栏导航

侧边栏按功能组织，包含 13 个主要分类，共 26 个实用工具：

1. **编码工具** - JSON格式化、URL编码/解码
2. **文本处理** - 大小写转换、文本统计
3. **加密工具** - MD5/SHA哈希、Base64编解码
4. **时间工具** - 时间戳转换、时区转换
5. **颜色工具** - 颜色选择器、HEX/RGB转换
6. **网络工具** - IP查询、端口扫描
7. **正则表达式** - 测试工具、生成器
8. **图片处理** - 图片压缩、格式转换
9. **单位转换** - 长度、重量、温度等转换
10. **数据生成** - 假数据生成、UUID生成
11. **代码工具** - 代码美化、格式化
12. **系统工具** - 系统信息、进程管理
13. **开发者工具** - API测试、环境变量管理

侧边栏支持折叠和展开，点击分类名称可切换显示/隐藏该分类下的工具：

![侧边栏导航](../screenshots/global/sidebar-navigation.png)

## 搜索功能

使用全局快速搜索功能可以快速定位工具：

- **快捷键**: `Ctrl+K` (Windows/Linux) 或 `Cmd+K` (macOS)
- **功能**: 输入工具名称或关键词即可快速跳转到对应工具
- **界面**: 搜索框会覆盖在主界面上，不影响当前操作状态

![搜索功能](../screenshots/global/search-function.png)

## 主题切换

DevToolkit 支持亮色和暗色两种主题模式：

- **快捷键**: `Ctrl+Shift+T` (Windows/Linux) 或 `Cmd+Shift+T` (macOS)
- **手动切换**: 点击右上角的主题切换按钮
- **自动适应**: 系统会自动检测并跟随系统主题设置

![主题切换](../screenshots/global/theme-switch.png)

## 工具页面通用结构

每个工具页面都遵循统一的布局模式，包含以下四个主要区域：

### 1. 选项配置面板
位于工具页面顶部，提供该工具的专属配置选项：
- JSON工具：缩进大小、格式化选项等
- 颜色工具：颜色模式选择、透明度设置等

![选项面板](../screenshots/global/options-panel.png)

### 2. 输入区域
位于页面左侧，提供文本输入框或文件上传功能：
- 支持多行文本输入
- 支持拖放文件
- 实时输入验证（如JSON语法检查）

![输入区域](../screenshots/global/input-area.png)

### 3. 输出区域
位于页面右侧，显示处理结果：
- 格式化后的输出
- 错误信息（如有）
- 复制按钮（一键复制）

![输出区域](../screenshots/global/output-area.png)

### 4. 历史记录
位于页面底部，保存最近的使用记录：
- 显示操作时间和结果预览
- 点击可恢复历史状态
- 支持清空历史记录

![历史记录](../screenshots/global/history-section.png)

## 快捷键列表

| 快捷键 | 功能 | 平台 |
|--------|------|------|
| `Ctrl+K` / `Cmd+K` | 全局搜索工具 | 所有平台 |
| `Ctrl+B` / `Cmd+B` | 切换侧边栏显示/隐藏 | 所有平台 |
| `Ctrl+Shift+T` / `Cmd+Shift+T` | 切换主题（亮色/暗色） | 所有平台 |
| `Ctrl+Enter` / `Cmd+Enter` | 执行当前工具操作 | 所有平台 |
| `Ctrl+C` / `Cmd+C` | 复制选中内容或全部结果 | 所有平台 |
| `Ctrl+L` / `Cmd+L` | 清空输入区域 | 所有平台 |
| `Ctrl+H` / `Cmd+H` | 显示/隐藏历史记录 | 所有平台 |

快捷键可以在设置中自定义，个性化操作体验。
