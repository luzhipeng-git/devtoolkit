# 通用功能

所有工具都支持以下通用功能，确保一致的使用体验。

## 复制功能

每个工具都集成了一键复制功能，方便快速获取处理结果：

![复制按钮](../screenshots/global/copy-button.png)

**使用方式**：
- 点击输出区域右上角的"复制"按钮
- 支持复制全部结果或仅选中部分
- 复制成功后会显示提示动画

**技术实现**：
```typescript
// CopyButton 组件提供统一的复制功能
<CopyButton text={outputResult} />
```

## 历史记录

所有工具都会自动保存最近的操作历史，方便重复使用或回溯：

![历史记录](../screenshots/global/history-section.png)

**功能特性**：
- 自动保存：每次执行操作后会自动保存
- 时间戳显示：精确到秒的操作时间
- 结果预览：显示输入内容和输出摘要
- 一键恢复：点击历史记录可恢复到该状态
- 批量管理：支持清空历史记录

**数据存储**：
- 使用 localStorage 持久化存储
- 单个工具最多保存 50 条记录
- 跨会话保持历史记录

## 交换按钮

在需要双向转换的工具中（如URL编解码、大小写转换等），提供了交换功能：

![交换按钮](../screenshots/global/swap-button.png)

**使用场景**：
- URL 编码 ↔ 解码
- 大写 ↔ 小写转换
- HEX ↔ RGB 颜色转换
- 时间戳 ↔ 日期时间转换

**操作方式**：
1. 在输入框中填入原始数据
2. 点击执行按钮获得结果
3. 点击交换按钮，将结果作为新的输入
4. 继续进行反向转换

## 错误提示

当操作出现问题时，会显示友好的错误提示：

![错误提示](../screenshots/global/error-banner.png)

**错误类型**：
- **输入验证错误**：JSON格式错误、无效的颜色值等
- **处理错误**：网络请求失败、计算错误等
- **系统错误**：权限不足、存储空间不足等

**错误处理策略**：
- 即时反馈：输入时实时验证，避免无效操作
- 详细信息：显示错误原因和解决建议
- 优雅降级：部分功能失败时保持其他功能可用

## 选项面板

每个工具都提供专属的配置选项，通过选项面板进行设置：

![选项面板](../screenshots/global/options-panel.png)

**常见选项类型**：
1. **格式化选项**
   - JSON：缩进大小（2/4空格）、是否保留引号等
   - 代码：语言类型、缩进风格等

2. **转换选项**
   - 颜色：输出格式（HEX/RGB/HSL）、是否包含透明度
   - 时间：时区、时间格式等

3. **安全选项**
   - 敏感数据处理：是否记录历史、是否显示完整内容
   - 性能选项：是否启用缓存、超时时间等

**设置保存**：
- 使用 localStorage 持久化个人设置
- 支持导出/导入配置文件
- 重置为默认设置功能

## 使用技巧

### 1. 批量处理
- 使用文件上传功能批量处理多个文件
- 在输入区域中，每行一个条目，自动批量处理

### 2. 快捷操作
- 使用快捷键 `Ctrl+Enter` 快速执行
- 使用 `Ctrl+L` 快速清空输入
- 使用 `Ctrl+H` 快速切换历史显示

### 3. 结果管理
- 支持多次操作结果对比
- 可在不同标签页同时使用多个工具
- 历史记录支持搜索和过滤

### 4. 性能优化
- 大文件处理时会显示进度条
- 长时间操作会显示处理状态
- 支持中断正在进行的操作

这些通用功能确保了 DevToolkit 虽然工具多样，但操作体验始终保持一致和高效。
