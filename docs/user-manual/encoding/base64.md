# Base64 编解码

## 功能简介
字符串与 Base64 编码的相互转换。

## 编码模式
![Base64 编码](../screenshots/encoding/base64-encode.png)

### 操作步骤
1. 确保在「编码」标签页
2. 在输入区域输入文本
3. 结果自动显示

### 参数说明
| 参数 | 说明 | 可选值 |
|------|------|--------|
| 字符集 | 编码使用的字符集 | UTF-8、ASCII |
| 换行 | 输出是否换行 | 无、64 字符、76 字符 |

## 解码模式
![Base64 解码](../screenshots/encoding/base64-decode.png)

### 操作步骤
1. 切换到「解码」标签页
2. 输入 Base64 字符串
3. 结果自动显示为原始文本
