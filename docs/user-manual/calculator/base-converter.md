# 进制转换

## 功能简介
在二进制、八进制、十进制、十六进制之间进行数值转换，支持批量转换。

![进制转换结果](../screenshots/calculator/base-converter-result.png)

## 操作步骤
1. 在选项面板选择「源进制」（十进制、二进制、八进制、十六进制）
2. 在输入框输入数字
3. 自动显示四种进制的结果

### 参数说明
| 参数 | 说明 |
|------|------|
| 源进制 | 输入数字的进制 |

### 自动前缀检测
当源进制选择「十进制」时，自动检测以下前缀：
- `0x` → 十六进制
- `0b` → 二进制
- `0o` → 八进制

### 结果显示
| 进制 | 前缀 | 示例 |
|------|------|------|
| 二进制 | 0b | 0b11111111 |
| 八进制 | 0o | 0o377 |
| 十进制 | 无 | 255 |
| 十六进制 | 0x | 0xFF |

### 批量转换
![批量转换](../screenshots/calculator/base-converter-batch.png)
支持多行输入，每行一个数字，批量转换所有结果。
